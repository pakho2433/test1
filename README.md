# 星光跳跳島

一個可直接在手機及電腦瀏覽器遊玩的 Three.js 3D 平台跳躍遊戲。

## 遊戲玩法

- 使用畫面方向鍵或鍵盤 WASD／方向鍵移動。
- 按「跳躍」或空白鍵跳上平台。
- 收集 5 顆星星後，進入終點光圈過關。

## GitHub Pages

部署工作流程已放在 `.github/workflows/pages.yml`。

如網站未自動開啟，請到 Repository `Settings` → `Pages`，將 `Build and deployment` 的 Source 設定為 `GitHub Actions`，然後重新執行工作流程。
